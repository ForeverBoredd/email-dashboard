Setup

  1. For the easiest school setup, use Outlook Desktop if the desktop app is already signed in.
  2. Create a Microsoft Entra app registration for a single-page app.
  3. Add http://localhost:8787 as the redirect URI.
  4. Allow delegated Graph permissions: User.Read, Mail.Read, and Mail.ReadWrite.
  5. Paste the client ID here and sign in.

Draft replies are created in Outlook and are never sent automatically.
